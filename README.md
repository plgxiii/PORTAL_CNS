# PORTAL_CNS
Projeto desenvolvildo como uma forma de gerenciar estoque em todas as áreas, chegada do produtos,controle de entrada e saída e e gestão de Pedidos com um painel dashbord que lá da para acompanhar os produtos que estão para vencer, os produtos que mais estão saindo e para onde está saindo e muito mais. O projeto foi desenvolvido inteiro em MVC C#
